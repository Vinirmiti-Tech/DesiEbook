# Book Reader

Book Reader simple UI. Based on FBReader (https://github.com/geometer/FBReaderJ):

Supported formats: fb2, html, txt, epub, mobi, pdf, rtf, djvu.

All the best in this world is free!

# Manual install

    gradle installDebug
